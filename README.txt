# Sistema Web para Monitoramento de Ensaios Laboratoriais

Projeto desenvolvido para o Projeto Integrador do curso de Ciência de Dados da UNIVESP.

## Tecnologias utilizadas

- Python
- Flask
- SQLite
- pandas
- HTML/CSS

## Funcionalidades

- Cadastro de amostras
- Banco de dados SQLite
- Dashboard de indicadores
- Exportação para Excel
- Cálculo automático de tempo de processamento
